# 💳 UPI Transaction Analysis Dashboard (Power BI)

## 🚀 Project Overview

This project presents an interactive **UPI Transaction Analysis Dashboard** built using Power BI to analyze transaction behavior, customer trends, and financial patterns across multiple dimensions.

The goal is to transform raw transaction data into **meaningful business insights** that can help in decision-making.

---

## 📊 Key Business Questions Answered

* How do transaction balances vary month-wise?
* Which cities contribute the most to transactions?
* What are the trends in remaining account balance?
* How do payment methods and transaction types impact behavior?

---

## 🛠️ Tools & Technologies

* **Power BI Desktop**
* **Power Query (Data Cleaning & Transformation)**
* **DAX (Data Analysis Expressions)**
* **Excel (.xlsx dataset)**

---

## 📂 Dataset

* Source: UPI Transactions Dataset
* Data includes:

  * Transaction Amount
  * Customer Age & Gender
  * City & Currency
  * Payment Method & Type
  * Remaining Balance

---

## ⚙️ Data Preparation

* Cleaned and transformed raw data using **Power Query**
* Handled missing/null values
* Created calculated columns for better segmentation
* Structured dataset for efficient analysis

---

## 🧠 DAX Measures Used

* Total Transaction Amount
* Remaining Balance Calculation
* Monthly Aggregations
* Custom metrics for trend analysis

---

## 📈 Dashboard Features

### 🔹 Interactive Filters

* Bank Name (Sent/Received)
* City
* Device Type
* Gender
* Payment Method
* Transaction Type

### 🔹 Visualizations

* 📉 Monthly Balance Trend (Line Chart)
* 📊 City-wise Transaction Analysis
* 📋 Detailed Matrix View (City + Month breakdown)

## 🔍 Key Insights

* 📈 Peak balance observed in **June (~8.5K)** indicating high transaction activity
* 📉 Significant dip in **May (~8.2K)** suggests reduced activity
* 🏙️ Cities like **Mumbai & Delhi** show consistently high transaction volumes
* 💰 Remaining balance trends indicate stable financial behavior across months
